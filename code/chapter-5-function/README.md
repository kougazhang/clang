## 5.5 内部函数与外部函数

内部函数：一个源文件只能在本文件内被调用。

定义内部函数的方法：

```
// 这里的 static, 不是指存储方式，指函数只能在本文件调用
static type functionName()
```

外部函数：外部函数在整个源码中有效。

定义外部函数的方法：

```
extern type functionName()
```

如果在函数定义中没有说明 `extern` 或 `static`, 则默认为 `extern`.
